# Тест «Хозяин встречи»

Готовый одностраничный тест для GitHub Pages.

Что внутри:
- открытые вопросы без подсказок;
- вопросы с одним вариантом ответа;
- вопросы с несколькими вариантами ответа;
- возможность пропускать вопросы;
- опция перемешивания вопросов перед началом;
- итоговая обратная связь по темам.

Для публикации:
1. Загрузить `index.html` в репозиторий GitHub.
2. Включить GitHub Pages для ветки и папки, где лежит файл.
3. Дождаться публикации страницы.

Примечание:
- `index.html` — самодостаточный файл, его достаточно для работы сайта.
- `app.js` приложен отдельно как копия логики теста на случай, если захочется потом вынести скрипт из HTML.
