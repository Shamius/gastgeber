# Тест «Стань хозяином встречи»

В папке лежит готовый `index.html` для публикации на GitHub Pages.

## Что внутри
- открытые вопросы без подсказок во время прохождения;
- вопросы с одним вариантом ответа;
- вопросы с несколькими вариантами ответа;
- итоговая обратная связь по общему результату и по темам.

## Как опубликовать на GitHub Pages
1. Создай новый репозиторий на GitHub.
2. Загрузи в корень репозитория файл `index.html`.
3. В настройках репозитория открой раздел **Pages**.
4. В качестве источника выбери ветку `main` и папку `/root`.
5. Сохрани настройки — через короткое время GitHub выдаст ссылку.

## Основа теста
Тест посвящён теме сильной и управляемой встречи.
